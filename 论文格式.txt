
摘要部分：
{
    “摘要”：格式与一级标题同，（毕业设计规范化要求中给出的论文样稿里，摘要两字中间空了三格）
    “关键词：”：顶格，小四号，宋体，加粗
     关键词之间用逗号隔开，摘要部分其他文字都与正文格式相同
}

Abstract：
{
    采用Times New Roman字体，其他与摘要类似
    “Abstract”：格式与一级标题同，注意字体是Times New Roman
}

**
一级标题：单独起页，居中，小三号，宋体加粗
二级标题：左顶格，四号，宋体加粗
三级标题：左顶格，小四号，宋体加粗

正文：小四号，宋体/Times New Roman（注意英文字体，默认的正文英文字体是Calibri，可以在“正文”样式中修改） 1.25倍行距
**

以上标方式引用参考文献

图：
{
    图中文字：小五号，宋体
    图号：（居中，）小五号，宋体，放在图的下方
    空行：图前，图和图号中间，图号后，总共三个空行
    别忘记引用图！
}

表：
{
    表号：（居中，）小五号，宋体，放在表上方
    表内文字：小五号，宋体
    表注：六号，宋体
    空行：表（包括表号，表注）前后各一个空行，总共两个空行
    别忘记引用表！
}

致谢：
{
    “致谢”：格式与一级标题同
    文字格式与正文同
}

参考文献：
{
    “参考文献”：格式与一级标题同
    文字格式与正文同
    按引用顺序排列
}

附录 英文缩写词


目录部分的格式最后调，先用自动生成的，完稿的时候再修改目录格式
{
    “目录”：居中，小三号，宋体加粗（论文样稿里，目录两字中间空了四格）
    目录项：小四号，宋体
    页码：摘要和Abstract的页码用希腊数字，正文页码用阿拉伯数字
}
